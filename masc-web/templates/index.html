MASC WEB

Welcome to MASC Web! Get started with Deploy and get an overview of the framework with Help. If you would like to read the papers associated with the project or contact the creator you can find this information in the About section.

Getting Started

Download the project from HERE to get started. If you would like to use IntelliJ to run the project (like we recomend) go to HERE. Make sure to select the community version of IntelliJ. Once both the project is installed and IntelliJ, follow the intructions found in Deploy.
